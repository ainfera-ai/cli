# ainfera

Command-line interface for the Ainfera platform — the neutral trust
and settlement layer for the agentic economy.

## Install

```bash
pip install ainfera
```

## Usage

```bash
ainfera register --name my-agent
ainfera skill-scan my-agent
ainfera deploy ./my-agent
```
